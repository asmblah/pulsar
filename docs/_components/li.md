## Options

Option | Type   | Description
------ | ------ | --------------------------------------------------------------
class  | string | CSS classes, space separated
id     | string | A unique identifier, if required
value  | string | The value to display
data-* | string | Data attributes, eg: `'data-foo': 'bar'`
