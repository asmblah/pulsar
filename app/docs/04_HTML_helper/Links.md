## Link

### Example usage:

	{{
		html.link({
			'href': 'http://www.google.com',
			'label': 'Google'
		})
	}}

### Options

Option  | Description
------- | ----------------------------------------------------------------------
class 	| (string) CSS classes, space separated
data 	| (hash) data attributes by key/value
href	| (string) The URL attribute
id 		| (string) A unique identifier, if required
label 	| (string) The link label
